# Build Order

## Phase 1: Working Money Loop

Build one thin vertical slice:

1. source discovery
2. normalization
3. enrichment
4. scoring
5. policy gate
6. listing creation
7. order capture
8. shipment tracking
9. profit measurement

## Phase 2: Operational Hardening

- duplicate detection
- inventory drift protection
- repricing rules
- event persistence
- retry and compensation logic

## Phase 3: Channel Hardening

- WhatsApp operational flow
- quote and invoice automation
- post-delivery follow-up

## Phase 4: Broader Autonomy

- more sources
- more categories
- richer agents
- broader optimization loops

## Rule

Do not start a new major subsystem until the current phase has one measurable business outcome.
